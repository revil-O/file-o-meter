---------------------------------
- After successful installation -
---------------------------------

Delete the following files from the servers filesystem:

	- install.php
	- update.php
	- FOM.sql

Remove write permission (CHMOD) from the following folders:

	- ./config/
	- ./web_services/access.wsdl
	- ./web_services/fom.wsdl

Create a .htaccess file for security restrictions for the following folders to prevent unauthorized access:

	- ./files/
	- ./config/

Some features of File-O-Meter (such as Auto-Backup) require a cronjob.
Please configure a cronjob which executes the file cj/cj.php four times per hour!


---------------------------------
- First steps in File-O-Meter:  -
---------------------------------

Menu item 'Basic setup'
	- Complete the configuration settings.
	- Add some document types, for instance 'Images', 'Sound files' et cetera.

Menu item 'Project management'
	- Add at least one project. Each project is used as root directory in the tree view.

Menu item 'Usergroup management'
	- Add more usergroups. It is recommended to add a group with limited access permissions for default users.

Menu item 'Useraccount management'
	- Add users


Now you are able to use File-O-Meter. You may:

	- Add folders and subfolders to your project(s)
	- Upload files
